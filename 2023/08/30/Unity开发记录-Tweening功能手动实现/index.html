Unity开发记录-Tweening功能手动实现

Tweening全称In-Betweening，是指处理两帧之间的平滑过渡。目前有DOTween插件可以快速创建In-Betweening

Tweening功能手动实现：

  1. MyCurveSample类：提供曲线的采样功能（给定x返回y）
  2. MyCurve类：用于描述曲线的类型、方向，提供静态的MyCurve类作为预设的曲线
  3. Procedures类：单例模式，提供CurveProcedure方法进行采样，并将采样的值（y）传入给定的Action中

常见的缓动曲线查询：

MyCurveSample类实现

  • curveFuncs：曲线字典，曲线类型对应Func委托，Func委托传入x值，返回y值
  • SampleCurve：采样函数，根据传入的曲线类型和x值返回y值
  • CurveType和CurveDir：提供两个枚举分别表示曲线类型和曲线方向

MyCurve类

用于描述曲线的类型、方向，提供静态的MyCurve类作为预设的曲线

Procedures类

单例模式，提供CurveProcedure方法进行采样，并将采样的值（y）传入给定的Action中

CurveProcedure方法的参数：

  • MyCurve myCurve：表示曲线的类型
  • float time：表示x从0到1所需要的时间
  • Action<float> action：对于每一个采样到的y值，将其作为参数传入此action中
  • Action onComplete = null：默认参数，在x到达1之后调用，为null则不调用
  • string id = ""：默认参数，用于标识
