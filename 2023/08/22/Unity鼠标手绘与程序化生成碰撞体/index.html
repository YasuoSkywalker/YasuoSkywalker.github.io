Unity鼠标手绘与程序化生成碰撞体

需求分析

  • 实现鼠标当画笔拖拽画线功能（Draw and Guess的效果）
  • 画出线后，为线条自动添加多边形碰撞体

实现思路

线条的生成

  1. LineRenderer组件可以帮助渲染线条
    • Position储存了线条的各个点的坐标 ( Vector3 )
    • 检测到鼠标按下时，记录鼠标的位置，使用SetPosition添加到线条的点集中
  2. 使用两个变量记录前一个点的位置和当前鼠标的位置，用于判断两点距离是否足够远，足够远的话就将鼠标位置加入Position点集

碰撞体的生成

  1. PolygonCollider2D组件
  2. 利用向量叉积，根据线条各点之间形成的向量 ( 表示方向 ) 求出多边形碰撞体的点集
    • PolygonCollider2D将点集中的点按顺序首尾相连形成碰撞体的形状
    • ( 0,0,1 )与线条方向向量的叉积求出点的左右平移方向，再平移线条宽度的一半即可求出碰撞体的点集

计算出平移方向后（根据上式得到往左边的平移方向），平移得到左右边缘的点，然后加入点集中。

注意加入点集的顺序，左边的边缘点加到0，右边的边缘点加到末尾，才可以保证顺序正确（保证了首尾相接）
