Unity粒子系统学习记录6-生命周期模块

发表于2023-07-20|更新于2023-07-20|Unity学习记录
|阅读量:

Unity粒子系统学习记录6-生命周期模块

Velocity over Lifetime

控制粒子在其生命周期内的速度

Linear

粒子在 X、Y 和 Z 轴上的线性速度。

Space

指定 Linear X, Y, Z 轴是参照本地空间还是世界空间。

Orbital X, Y, Z

设置粒子围绕 X、Y 和 Z 轴的轨道速度。

Offset X, Y, Z

轨道中心的位置，适用于轨道运行粒子。

Radial

粒子远离/朝向中心位置的径向速度。负数为朝向中心，正数为远离中心

Speed Modifier

在当前行进方向上/周围向粒子的速度应用一个乘数。

Limit Velocity over Lifetime module

此模块控制粒子的速度在其生命周期内的限制。

Separate Axes

将轴拆分为单独的 X、Y 和 Z 分量。

Space

选择速度限制是适用局部空间还是世界空间。仅当启用了 Separate Axes 时，此选项才可用。

Speed

设置粒子的速度限制。

Dampen

当粒子速度超过速度限制时，粒子速度降低的比例。

Drag

对粒子速度施加线性阻力（可用于模拟空气阻力）。

Multiply by Size

启用此属性后，较大的粒子会更大程度上受到阻力系数的影响。

Multiply by Velocity

启用此属性后，较快的粒子会更大程度上受到阻力系数的影响。

Force over Lifetime module

粒子在生命周期内受到力的影响，通过此模块可以模拟吸力、风力对粒子的加速等效果

X, Y, Z

在 X、Y 和 Z 轴上施加到每个粒子的力。

Space

选择是在局部空间（local）还是在世界空间（world）中施力。

Randomize

使用 Two Constants 或 Two Curves 模式时，此属性会导致在每个帧上在定义的范围内选择新的作用力方向。因此会产生更动荡、更不稳定的运动。

Color over Lifetime module

指定粒子的颜色和透明度在其生命周期中如何变化。

Color

粒子在其生命周期内的颜色渐变。渐变条的左侧点表示粒子寿命的开始，而渐变条的右侧表示粒子寿命的结束。

Color by Speed module

设置粒子的颜色根据粒子速度（每秒的距离单位）变化。

Color

在速度范围内定义的粒子的颜色渐变。

Speed Range

颜色渐变映射到的速度范围的下限和上限（超出范围的速度将映射到渐变的端点）。

Size over Lifetime 模块

根据生命周期改变粒子大小，如水中冒出的气泡

Separate Axes

在每个轴上独立控制粒子大小。Z 仅用于网格粒子。

Size

通过一条曲线定义粒子的大小在其生命周期内如何变化。

Size by Speed

根据速度（每秒的距离单位）改变粒子的大小，此模块不应与 Limit Velocity Over Lifetime 模块一起使用，除非希望粒子在减速时改变其大小。

Separate Axes

在每个轴上独立控制粒子大小。

Size

通过曲线定义粒子在速度范围内的大小。

Speed Range

大小曲线映射到的速度范围的下限和上限（超出范围的速度将映射到曲线的端点）。

Rotation over Lifetime

控制粒子在移动时的旋转。

Separate Axes

允许根据每个轴指定旋转。启用此选项后，即可为 X、Y 和 Z 轴中的每个轴设置旋转。

Angular Velocity

旋转速度（以度/秒为单位）

Rotation by Speed module

设置粒子的旋转根据粒子速度（每秒的距离单位）变化，例如滑坡的岩石（粒子表示在地面上移动的固体对象）。

Separate Axes

为每个旋转轴独立控制旋转。

Angular Velocity

旋转速度（以度/秒为单位）。

Speed Range

大小曲线映射到的速度范围的下限和上限（超出范围的速度将映射到曲线的端点）。
