Unity粒子系统学习记录1-概述与模块简介

两种解决方案

区别 Particle System Visual Effect Graph（VFX Graph）
基于 cpu gpu
粒子数量 数千 数百万
渲染管线 支持全部渲染管线 不支持内置渲染管线
物理系统 可以与Unity物理系统交互 与特定元素交互
使用 模块化、修改参数 节点可视化、修改节点参数

Particle System系统结构

  • Particle System组件+内部分模块的形式

Particle System模块简介

模块概述：

模块 功能
main 控制粒子的初始状态、全局状态
Emission 控制粒子的发射速率、时间、波次
Shape 定义发射粒子体积的形状
Velocity over Lifetime 控制粒子在其生命周期内的速度
Limit Velocity over Lifetime 在粒子生命周期内限制粒子的速度
Inherit Velocity 控制粒子的速度如何随时间推移而受到其父对象移动的影响
Lifetime by Emitter Speed 根据粒子生成时发射器的速度控制每个粒子的初始生命周期
Force over Lifetime 指定力对粒子产生影响
Color over Lifetime 指定粒子的颜色和透明度在其生命周期中如何变化
Color by Speed 指定粒子的颜色根据速度变化
Size over Lifetime 控制粒子在其生命周期的大小
Size by Speed 控制粒子大小根据粒子速度变化
Rotation over Lifetime 控制粒子在其生命周期的旋转
Rotation by Speed 控制粒子的旋转根据粒子速度变化
External Forces 修改力场对系统发射的粒子的影响
Noise 为粒子添加湍流（使粒子运动不规律）
Collision 控制粒子与场景其他游戏对象的碰撞
Triggers 控制粒子的触发器
Sub Emitters 在某粒子生命周期的阶段创建附加粒子发射器
Texture Sheet Animation 控制粒子的动画帧播放
Lights 控制粒子的实时光照
Trails 控制粒子的轨迹（拖尾效果）
Custom Data 在Editor中定义要附加到粒子的自定义数据格式
Renderer 设置粒子的图像或网格如何被其他粒子变化、着色和控制
