创建并编辑骨骼

点击Open Sprite Editor，进入编辑页面，左上角选择skinning editor

​ Unity将每个图层都分为多个多边形，每个多边形可以受到骨骼的影响，当骨骼运动时这些多边形也会跟着运动。通过调整多边形的顶点、边缘以及多边形对每条骨骼的权重可以制作精细的动画

​ 点击create bone，用鼠标绘制骨骼，当某条骨骼需要以另一条骨骼为基准时，先点击基准骨骼，然后进行绘制

​ 创建完骨骼后，点击Auto Geometry自动创建顶点

​ 之后通过Sprite influence调整骨骼影响的图层，edit geometry调整顶点，进行微调

Sprite Editor面板：

  • preview pose：预览骨骼绑定效果
  • restore pose：重置pose
  • edit bone：编辑已有的骨骼
  • create bone：创建骨骼
  • splite bone：将已有的骨骼一分为二
  • auto geometry：自动生成多边形
  • edit geometry：编辑每个多边形的顶点
  • create vertex：创建新的多边形顶点，用于使动画更加精细
  • create edge：创建新的多边形边缘
  • splite edge：分割边缘
  • auto weights：自动赋予权重
  • weight slider：编辑多边形受骨骼影响的权重
  • weight brush：权重刷
  • bone influence：编辑骨骼影响的多边形
  • sprite influence：编辑多边形所受影响的骨骼
  • copy rig：复制当前骨骼
  • paste rig：粘贴
