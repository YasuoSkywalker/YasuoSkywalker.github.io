Unity粒子系统学习记录11-ExternalForce模块和粒子系统力场

External Force模块

可以修改风区和粒子系统力场对粒子的影响

属性 功能
Multiplier 应用于风区外力的比例值。
Influence Filter 选择是否基于层遮罩 (Layer Mask) 或通过列表 (List) 包含力场。
List 定义可影响此粒子系统的力场的显式列表。当 Influence Filter 设置为 List 时，将显示此属性。
Influence Mask 使用层遮罩来确定哪些力场影响此粒子系统。当 Influence Filter 设置为 Layer Mask 时，将显示此属性。

粒子系统力场

将力应用于粒子系统中的粒子，在粒子系统中启动External Force，指定Layer Mask或者指定Force Field组件。

Shape

属性 功能
Shape 选择力场的形状
Start Range 设置力场形状内部从何处开始
End Range 设置力场形状外部从何处结束

Direction

  • 设置线性力以应用于沿 x 轴、y 轴和 z 轴的粒子,值越高，力越大

Gravity

设置引力

属性 功能
Strength 设置吸引力，值越高力越大
Focus 设置引力中心点

Rotation

设置力场的涡流

属性 功能
Speed 涡流的速度，值越高，粒子随涡流的速度越快
Attraction 粒子在涡流中的向心力，值为1为最大吸引力，值为0则不应用吸引力
Randomness 设置形状的随机轴以推动粒子四处移动。值为 1 表示最大随机性，值为 0 则不应用随机性

Drag

阻力

属性 功能
Strength 设置阻力的强度，以减慢粒子的速度。值越高，强度越大
Multiply by Size 是否根据粒子的大小调整阻力的强度
Multiply by Velocity 是否根据粒子的速度调整阻力的强度

Vector Field向量场

预先计算的力场，为粒子提供速度。
