Unity粒子系统学习记录2-主模块

Duration

  • 粒子系统的持续时间

Looping

  • 是否循环

Prewarm

  • 是否预加载，勾选后，系统在初始化时就像已经完成一个周期一样

Start Delay

  • 粒子的发射延迟

Start LifeTime

  • 粒子刚生成时的生命周期时长

Start Speed

  • 粒子发射时的初始速度

3D Start Size

  • 如果要分别控制每个轴的大小，启用这个属性

Start Size

  • 粒子初始大小

Flip Rotation

  • 使一些粒子在生成时以相反的方向旋转（0~1之前的值）
  • 值代表相反方向旋转的粒子所占总粒子的比例

Start Color

每个粒子的初始颜色（需要在Renderer里使用粒子材质）

  • color：指定初始颜色
  • Gradient：指定渐变范围，在这个范围内随机取色
  • 其他选项是基于以上两种选项

Gravity Modifier

  • 粒子的重力

Simulation Space

粒子参照的坐标

  • Local：局部坐标（粒子的运动是相对发射器）
  • World：世界坐标（粒子的运动是相对世界，如汽车轮胎的扬尘效果）
  • Custom：自定义参照的物体

Simulation Speed

  • 整个粒子系统更新的速度

Delta Time

  • 控制粒子系统是否受到Time类中Scale的影响

Scaling Mode

粒子的缩放模式

  • Hierarchy：跟随父级缩放
  • Local：跟随粒子系统的Transform的Scale缩放
  • Shape：将缩放应用于粒子的起始区域，但不影响粒子大小

Play On Awake

  • 是否在被创建时就启动粒子系统

Emitter Velocity Mode

粒子系统如何计算粒子的速度

  • rigidbody：通过Rigidbody组件
  • transform：如果没有Rigidbody，则默认使用transform
  • custom：常量

Max Particle

  • 允许同时存在的最大粒子数量

Auto Random Seed

  • 粒子发射随机化，取消勾选后可以设置随机数种子

Stop Action

所有粒子生命周期结束后，粒子系统要做的事

  • Disable：禁用粒子系统
  • Destroy：摧毁
  • Callback：回调，将OnParticleSystemStopped 回调发送给附加到游戏对象的任何脚本。

Culling Mode

当粒子特效不在摄像机范围内时如何处理

  • Automatic：自动模式，如果粒子系统是单词播放则不暂停，循环播放则暂停模拟
  • Pause And Catch-Up：暂停，但下次注视时会演算出它应该在的位置
  • Pause：暂停
  • Always Simulate：总是模拟

Ring Buffer Mode

保持粒子直到最大粒子数，粒子不会因为生命周期结束而被移除，而是在到达粒子最大数时会被作为新的粒子循环使用

  • Pause Until Replaced：粒子生命周期结束后暂停旧粒子，直到达到限制，此时粒子系统进行再循环，旧粒子显示为新粒子
  • Loop Until Replaced：粒子生命周期结束后倒回其生命周期的指定比例，直到达到限制，此时粒子系统进行再循环，旧粒子显示为新粒子
