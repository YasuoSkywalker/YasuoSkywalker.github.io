引用做函数的参数（引用传递）：

将引用作为参数传递到函数中，可以起到和指针传递一样的效果，即形参会修饰实参。

引用有以下几点注意事项：

1.引用必须初始化

2.引用一旦初始化就不可更改

3.不要返回局部变量的引用。因为它存在栈区之中。

4.引用的本质是一个 指针常量 ，其指向的值可以修改，但指针的方向不可以修改。

拓展：常量引用： const int &a

作用：一般用于修饰形参，防止误操作
