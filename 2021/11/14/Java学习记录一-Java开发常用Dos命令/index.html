Java学习记录一：常用Dos命令

Java开发常用Dos命令一：进入目录并操控文件

1.cd：进入目录

​ 例子：

​ cd d: (进入d盘)

​ cd music：进入music文件夹

​ 其他附加指令：

​ cd..（返回上一级）

​ cd/（返回根目录）

​ 示例：

​

2.dir：

查看当前所在目录的文件和文件夹

  • /s：查看当前目录及其子目录的文件和文件夹
  • /a：显示隐藏和非隐藏文件
  • /ah：只显示隐藏文件
  • /w：一行只显示五个（紧凑）
  • /p：以分页显示

3.cls：

清屏

4.type 文件名：

显示文本文档的内容
