Java学习记录-随机数

一、导包

  • import java.util.Random;

二、创建对象

  • Random x=new Random();

三、获取数据

  • int a=x.nextInt(100)+1;

注意：这里的括号内表示的是数据范围。**从0到括号内的数，区间左闭右开，即[0,100)**。加一则是整个区间加一，即[1,101)
